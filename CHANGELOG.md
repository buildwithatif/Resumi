# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-08

### Added
-   **Core Engine**: Resume parsing (PDF/DOCX), profile extraction, and job matching logic.
-   **Job Discovery**: Integration with RSS feeds (WeWorkRemotely, RemoteOK) and smart search URL generation.
-   **UI**: Dashboard for uploading resumes and viewing recommendations.
-   **Documentation**: Professional README, Contributing Guidelines, License, and Security Policy.
-   **CI/CD**: GitHub Actions workflow for automated testing and linting.

### Security
-   Removed sensitive `.env` files from git history.
-   Added `SECURITY.md`.
